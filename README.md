# Libro-ACA-2026
Repositorio para el libro de notas para los cursos de Aprendizaje y Comportamiento Adaptable
El repositorio incluye los archivos de los capítulos editados en quarto, correspondientes a una segunda edición del libro de notas para los cursos de Aprendixaje y Comportamiento Adpatable. Incluye tambien el código de los simuladores adicionales elaborados para estas notas, así como las figuras y videos correspondientes usados en las notas.
El material del *libro anterior* se encuentra en https://arbouria.github.io/Notas-Aprendizaje-y-Comportamiento-Adaptable-I/.
El autor del lbro es Arturo Bouzas y su página se encuentra en: https://www.bouzaslab25.com/
